# A Single Page Static React App

This  is Travel App made using React.js

## How to use ?

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

```sh
https://github.com/tanvirakibul/react-travel-app.git
```

Install it and run:

```sh
npm install
npm run dev
```
